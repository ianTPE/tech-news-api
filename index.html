Tech News API

提供 The Verge 最新文章的 Serverless API，適用於 GPT Actions / 商機分析。

Endpoint

GET /api/theverge/latest

  • limit：最多 50，預設 20，例如：?limit=10
  • since：ISO 日期時間，例如：?since=2025-01-01T00:00:00.000Z

範例：
/api/theverge/latest?limit=10
/api/theverge/latest?limit=15&since=2025-01-01T00:00:00.000Z

回應格式

{
  "source": "The Verge",
  "fetched_at": "2025-01-01T12:34:56.000Z",
  "articles": [
    {
      "title": "...",
      "link": "https://www.theverge.com/...",
      "published": "2025-01-01T11:22:33.000Z",
      "summary": "...",
      "source": "The Verge"
    }
  ]
}
